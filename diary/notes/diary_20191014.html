# 20191015日记

温度：8℃~15℃ 天气：晴

今晚又是12点之后开始写日记，似乎自己从来不想再12点之前停止打游戏，一直还是这样，骂了自己半天，对于游戏还是充满期待，可能自己对于诱惑还是难以抵制吧。

# 离职

今天去公司办理了离职，比较惊喜，又添了半个月的合同，还能拿半个月的工资，明然姐在那里给我开玩笑说让大家给我找个对象，结果那个姐真的说到了李琳，那样的女孩，我想想还是算了，见了面小鹿乱撞，现在想想她的样子就会觉得激动，追不了，配不上。帮明然姐电脑下了个ps2019，实在是跑不动，还是有空给她装个ps数字系列。

# 杂事

现在想想，感觉每天过的都是杂事，自己的项目没有前进半点，拖延症极其严重，除了打游戏，就是杂事把生活填满了。现在写日记，似乎才知道自己每天在干嘛。

  1. 填写学年鉴定表。
  2. 帮何肖修证件照。
  3. 帮袁光明学长整理一个文档，发现是分段线的事，没办法处理页码。
  4. 管少维通过验证消息问我见没见她的答案，似乎想和我说话，又是套路。

濮阳的妖风，早晚让我冻成冰棍。
