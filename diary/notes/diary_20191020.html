# 20191021日记

温度：12℃~23℃ 天气：阴

我快疯了，半夜十二点多的时候不知道为什么就是特别想打游戏，一到晚上就是想玩，控制不了自己，那种欲望，那种渴求不满，几乎把我彻底给坠落进去，我现在在考虑要不要远离宿舍，逃出自己的舒适区，不知道图书馆有没有可以给电脑充电的地方，明天我要去看一看，在这么下去，迟早死掉，梦想死掉，未来死掉。

# 卸载永雄联盟

就在我现在正在打字的时候，我实在忍受不了英雄联盟的诱惑，再这样下去，实在不敢想象未来会怎么样，我需要找一种代替方法，控制自己的欲望。项目结束之前，再下载游戏，我就是傻子，没有智商的傻子。如果连这点诱惑都控制不了，未来还能成什么大事，专科生看不起他们，他们还有周末搞兼职的，我这可好，周末白天不睡，晚上打游戏。我现在连个小学生都不如，上课不听，下课打游戏，我不就是我人生中最恶心的那一类人么，我现在居然和他们一样，我居然变成了我认为的傻子。

# 与肉饼店老板娘的闲谈

听了一则关于他们吵架的故事。

那天俩人都是忙活，嫂子照顾发烧的儿子，老板自己腰疼的受不了，嫂子手机居然还丢了，几件事聚合到一起，战争就突然间爆发了，没想到我哥说我嫂子：“你都不关心我！”像个小孩子一样，这让我觉得我之前有些观点是错的吧，真的每个人心中都有一个小孩子，都渴望被像小孩子一样照顾，虽然我不喜欢，可能我也是嘴上说说吧，谁知道呢，我也不想去装成小孩子，我也不希望被人照顾，我只想找人帮忙而已。

乐乐还是学习的问题，很聪明的小孩子，但是对于学习似乎并不喜欢，没有那种求知的感觉，全程都是被逼迫着学习，不过应该也是感觉到了父母挣钱的不容易，还是有一点点的学习主动性。希望他能好好发展，未来可期，懂礼貌，知礼节。

# 杂事

  1. 洗澡。
  2. 抢单买了衣柜伸缩架，还没到，发胶和手表到了，手表没有一个商标但是会走。
  3. 帮吕老师的朋友修了几张图，每次都是有少字多字的错误，我这毛病能陪我到老。
