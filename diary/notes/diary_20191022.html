# 20191023日记

温度：12℃~22℃ 天气：阴

今天帮平顶山的那个学弟带了份饭，走不长久，顾虑太多，分不清真假，难开窍，我发现我的世界是阴暗的，没有一个我十分值得去敬佩的人，我天天跟这群人呆在一起，真的怕最后和他们一起被社会淘汰，技术上没问题，但是没有一个情商在线的，这一群，唉~

# 罗小黑战记

今天周二，上了团课，看罗小黑战记，跟一群没怎么看过动画电影的人在一起看，他们欣赏不到电影中的美，看不懂电影，思想境界达不到，简称，垃圾观众，中国的影视圈、娱乐圈不就是被这群人搞坏的，娱乐圈成了颜值圈，没唱出来几首歌，没拍过几场电影的人现在大红大紫，真正有实力的无人问津，垃圾蔡徐坤能拿华语男歌手，真的是当今娱乐圈最大的悲哀。影视圈也是这样，什么肖战王一博什么的我只停留在听见一群无脑女粉丝说过，不知道长什么样子，不知道拍过什么电视剧，不知道有什么演技，都是一群垃圾，但是却稳坐流量顶峰，真的是垃圾。

# 瑞克和莫蒂

忽然间今天想找找这部动画片的资源，找了国内各大门户视频网站没找到，最后在人人影视上找到了，下载下来，太符合美国的风格了，全程血腥、色情，独特的视角解读瑞克，虽然口头上说自己成功的秘籍就是独来独往，但是看到和莫蒂的一朝一夕却煽情流泪，让人能看得进去，有人性的表面的逞强，也有人性内心的温柔，虽然俩人相爱相杀，但是也是互相依赖。浅浅的深度，恰到好处的幽默，让我一天追完一个季。

# 恶心的宣传册制作

不怕神一样的对手就怕猪一样的队友，真的是印证了，本来说的就是今天做不了，接着回了句下午三点半，真行，本来工作都推掉了，还能自己把自己捡起来，甲方给的介绍词还能自己再改动，以为自己文采多好，真的是醉了，都没有一点办事的能力么，让传个话也是各种难受，该传达的意思死活传达不到，真的是心累。逃了一节课，把东西做出来，还差点让我做了四页的东西变成两页，那不是要疯。唉，这批人，真的难带，赶紧考研吧，见见上层建筑。

# 2点之前睡觉get

现在12：48，室友还在打游戏，而我，要准备烧水休息，我就是这么看着别人在游戏中一步一步丧失生命。还有，好饿。
